# Taskfile

開発用の dev コンテナに task がインストールされているので、コンテナ内外で使用可能です。

## タスクの一覧

```sh
$ task
task: Available tasks for this project:
* default:        help
* help:           show all tasks
* install:        install dependencies
* lint:           ruff check and mypy
* lint:fix:       lint fix by ruff
* test:           test
```
